Controller Sprites

This is a collection of controller and input sprites meant to be used with the AGTD (Automatic GBG Texture Drawing) tool. Click on a sprite to copy its CSV data into your clipboard, which you can immediately paste inside the AGTD Arduino sketch file for importing.

What's AGTD?

AGTD (Automatic GBG Texture Drawing) is an Arduino program developed by Borri. Its purpose is to automatically draw any arbitrary image inside Game Builder Garage, using an Arduino microcontroller plugged into the Nintendo Switch. For more information, check out this great introduction and tutorial video by VideoDojo.

What if I don't own an Arduino?

If you still want to use those icons without having to buy an Arduino, you can enter the ID G 008 V7R L3P to download a game file containing all of them imported as textures. You can either use them as reference, or you can make a copy of the file and build your game there!
