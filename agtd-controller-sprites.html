About this tool

This is a collection of controller and input sprites for Game Builder Garage, specifically made for use with the Automatic Game Builder Garage Texture Drawing tool by Borri. AGTD is a program developed for Arduino devices, allowing them to automatically draw any image into a Texture Nodon within Game Builder Garage. For more info about AGTD, please check out this comprehensive introduction and tutorial video.

How to use

(This section assumes you know how to use AGTD.)

Each sprite is already pre-converted into the CSV data used by AGTD. Clicking on a sprite will copy its data to your clipboard, so you can immediately paste it into the AGTD sketch file.
